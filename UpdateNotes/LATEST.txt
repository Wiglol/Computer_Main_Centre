[ Network Toolkit — help 13 ]

[ Media Tools (FFmpeg) — help 14 ]

- convert 'file' to mp3/mp4/gif/wav/avi/mkv/flac/webm — convert between any format
- extract audio 'video.mp4' — pull audio from video
- trim 'file' start end — cut a clip
- resize 'file' WxH — resize video or image
- rotate 'file' 90/180/270 — rotate video
- volume 'file' 50%/200% — adjust audio volume
- compress 'file' — reduce file size (shows % saved)
- merge 'file1' 'file2' — concatenate media files
- media info 'file' — show duration, codec, bitrate, resolution
- thumbnail 'video' time — extract a frame as image
- Auto-detects FFmpeg even if not in PATH (searches common install locations)
