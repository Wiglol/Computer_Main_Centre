Computer Main Centre - Release Notes (05/03/2026)

[ UTF-8 Fix ]

- Fixed AI outputting garbled characters (e.g. "Ã¤" instead of "ä")
- All subprocess calls now use explicit UTF-8 encoding

[ Network Toolkit — help 13 ]

- New commands: ping, ip, dns, traceroute, netcheck, wifi, mobile, speedtest, net status, headers, flush dns
- 'wifi' shows SSID, signal, speed, channel, auth
- 'mobile' shows cellular/mobile broadband connection info
- 'speedtest' runs a real download speed test (shows Mbps)
- Existing 'ports' and 'kill' commands moved here from Automation

[ Media Tools (FFmpeg) — help 14 ]

- convert 'file' to mp3/mp4/gif/wav/avi/mkv/flac/webm — convert between any format
- extract audio 'video.mp4' — pull audio from video
- trim 'file' start end — cut a clip
- resize 'file' WxH — resize video or image
- rotate 'file' 90/180/270 — rotate video
- volume 'file' 50%/200% — adjust audio volume
- compress 'file' — reduce file size (shows % saved)
- merge 'file1' 'file2' — concatenate media files
- media info 'file' — show duration, codec, bitrate, resolution
- thumbnail 'video' time — extract a frame as image
- Auto-detects FFmpeg even if not in PATH (searches common install locations)

[ AI Manuals ]

- Fixed AI forgetting the = sign in macro/alias commands
- Added WRONG/RIGHT examples to all 3 manuals and system prompts
- Removed broken /qcount command from code and manuals
- Removed fake slash commands (/gitsetup, /gitlink, etc.) from medium manual

[ Help Sections Reorganized ]

- 13 = Network & Connectivity

[ Performance ]

- Removed 1.5s busy-wait on startup (update check)
- Reduced Ollama timeout 0.5s → 0.1s
- Lazy-loaded Java version detection (deferred until first use)
- Added pip dependency check sentinel (~/.ai_helper/.cmc_deps_ok) — skips on repeat starts
- Removed dead duplicate code (MIN_PY, REQUIRED vars, duplicate handlers)

[ Cross-Platform (Linux / Mac) ]

- New CMC_Platform.py module — centralizes all platform-specific logic
- Java detection now works on Linux (scans /usr/lib/jvm, /usr/java, ~/.sdkman, $JAVA_HOME)
- 'java set' writes to ~/.bashrc or ~/.zshrc on Linux/Mac instead of Windows registry
- Network commands (traceroute, wifi, mobile, flush dns, net status) all cross-platform
- sysinfo GPU detection uses lspci (Linux) or system_profiler (Mac)
- File/URL opening uses xdg-open (Linux) / open (Mac) automatically
- Arrow-key model/effort pickers work with ANSI escape sequences (not just msvcrt)
- 'cmd' command renamed to 'shell' — opens bash/zsh on Linux/Mac
- Project scaffolding generates .sh scripts on Linux/Mac instead of .bat
- Venv activation instructions adapt to platform automatically

[ Code Cleanup ]

- Removed duplicate command handlers (log, undo, echo)
- Removed duplicate Java helper functions
- Renamed bootstrap p() to _bootstrap_print() to avoid shadowing
- Consolidated platform checks into CMC_Platform.py imports
